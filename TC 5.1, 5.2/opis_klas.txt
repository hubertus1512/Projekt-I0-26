Admin:
  - Klasa potomona klasy User, urzytkowni zarządzający serwisem.
  Atrybuty:
  - assignedTickets : List<UUID>
  Metody:
  + banUser(user : User) : void
  + manageTags() : void
  + manageCategories() : void
  + resolveTicket(ticket : Ticket) : void

AccountStatus:
  - Enum zawierający statusy jakie może przyjmować konto.
  INACTIVE
  ACTIVE
  DEACTIVATED
  DELETED

Category:
  - Kategoria, którą można opisać stream.
  Atrybuty:
  - uuid : UUID
  - name : String

ChatMessge:
  - Wiadomość na chacie.
  Atrybuty:
  - messageId : int
  - senderUUID : UUID
  - content : String
  - sendDate : DateTime

Donation:
  - Klasa reprezentująca donacje wysłaną do streamera.
  Atrybuty:
  - uuid : UUID
  - amount : Decimal
  - senderUUID : UUID
  - receiverUUID : UUID
  - createdAt : DateTime

PlaylistComponent:
  - Klasa abstrakcyjna wymaga do osiagnięcia wzorca projektowego composite, który realizowany jest przez playliste.
  Metody:
  + display() : void

PlaylistComposite:
  - Klasa reprezentująca playliste oraz umożliwiająca zarządzanie nią.
  Atrybuty:
  - uuid : UUID
  - name : String
  - ownerUUID : UUID
  - accessType : AccessType
  - children : List<PlaylistComponent>
  Metody:
  + add(component : PlaylistComponent) : void
  + remove(component : PlaylistComponent) : void
  + display() : void

PlayslitLeaf:

  Atrybuty:

  Metody:
  + display() : void

  - endTime : DateTime

StreamerCard:

  - verified : boo

Streamer:
  - Klasa potomna User, urzytkownik, który stremuje.
  Atryuty:
  - followers : List<UUID>
  - subscribers : List<UUID>
  Metody:
  + startStream() : void
  + stopStream() : void

  + banUser(user : User) : void
  + requestCard() : void
  + withdraw() : void

Subscription:
  - Klasa przechowujaca inforamcje o subkrybcja urzytkowników do streamerów.
  Atrybuty:
  - uuid : UUID
  - amount : Decimal
  - userUUID : UUID
  - streamerUUID : UUID
  - billingDate : DateTime

Ticket:
  - Klasa przechowująca informacje o zgłoszonym problemie.
  Atrybuty:
  - uuid : UUID
  - reporterUUID : UUID
  - assignedAdminUUID : UUID
  - description : String  
  - status : TicketStatus
  - createdAt : DateTime
  - resolvedAt : DateTime

TicketStatus:
  - Enum opisujący stan ticketu.
  OPEN
  IN_PROGRESS
  CLOSED

User:
 - Klasa reprezentująca zalogowanego się użytkownika serwius.
 Atrybuty:
